# Convex MCP Server - Quick Start

## One-Click Setup

**[Click here to install](https://cursor.sh/settings?open=mcp&addServer=convex)**

## Or Run Setup Script

```bash
npm run setup:mcp
```

## Configuration

Add this to Cursor Settings → Features → MCP Servers:

```json
{
  "mcpServers": {
    "convex": {
      "command": "npx",
      "args": ["-y", "convex@latest", "mcp", "start"]
    }
  }
}
```

## Verify It Works

After restarting Cursor, ask the AI:
- "Show me all tables in my Convex deployment"
- "What functions are available?"
- "List environment variables"
